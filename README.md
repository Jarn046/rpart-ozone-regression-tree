# Regression Trees, Bagging and Random Forest (R)

## Description
This repository contains a series of experiments on tree-based methods in statistical learning.

The goal is to explore and compare:
- CART (Classification and Regression Trees)
- Bagging
- Random Forest

These experiments are part of my academic work.

## Context
This work is conducted in the context of my master's thesis on:
**Bagging and Random Forest in statistical learning**

## Methods
- Regression trees using `rpart`
- Model selection using cross-validation
- Pruning using cost-complexity
- Extensions to ensemble methods (Bagging, Random Forest)

## Dataset
- Ozone dataset (mlbench package)

## Objective
- Understand tree-based models
- Compare their performance
- Analyze bias-variance tradeoff
